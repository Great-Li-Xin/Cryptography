# Cryptography
> Powered by Flask
>密码学作业, 做的比较简陋

## Usage

``` bash
# Only support python3
git clone https://github.com/Handora/Cryptography

# install requirements
pip install -r requirements.txt

cd Cryptography

# run at localhost:8080
python safety_homework.py

```
